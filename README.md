# 开拓，星遗泰拉 ！

## 简介
[开拓，星遗泰拉 ！](https://steamcommunity.com/sharedfiles/filedetails/?id=3374564713)是一个为游戏添加拓展内容的Mod，内容以[崩坏：星穹铁道](https://sr.mihoyo.com/)为原型，添加了一系列遗器，旨在增强游戏的多样性和乐趣。玩家可以通过探索、挑战不同的环境和敌人，收集和强化遗器，以提升角色的能力和战斗效果。  
（小声bb：在去年我还发布过一个[星穹铁道mod](https://steamcommunity.com/sharedfiles/filedetails/?id=3212263371)（[mod展示视频1](https://www.bilibili.com/video/BV17q421w7za),[mod展示视频2](https://www.bilibili.com/video/BV1s5gueDEKq)），但是由于像素画太难画以及时间不是很充裕，于是便没有更新了，但是后面时间充裕了应该会把旧mod的内容搬进来并且进行优化）。

## 遗器介绍

打开背包后即可点击按钮![](StarRailRelic/Assets/Textures/RelicButton.png)装备遗器，或在控件中绑定按钮来快捷打开（默认为R)。

### 遗器获取

使用星轨通票![星轨通票](StarRailRelic/Content/Items/StarRailPass.png)以获取隧洞遗器（示例：![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Musketeer/MusketeerHead.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Musketeer/MusketeerHands.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Musketeer/MusketeerBody.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Musketeer/MusketeerFeet.png)），在肉后使用星轨专票![星轨专票](StarRailRelic/Content/Items/StarRailSpecialPass.png)以获取位面饰品（示例：![位面饰品示例](StarRailRelic/Content/Items/Relic/In/One/Space/SpaceSphere.png)![位面饰品示例](StarRailRelic/Content/Items/Relic/In/One/Space/SpaceRope.png)）。

星轨通票![星轨通票](StarRailRelic/Content/Items/StarRailPass.png)和星轨专票![星轨专票](StarRailRelic/Content/Items/StarRailSpecialPass.png)制作：10遗失晶块 ![遗失晶块](StarRailRelic/Content/Items/Miscellaneous/LostCrystal.png)

或者通过9金币制作燃料![星轨专票](StarRailRelic/Content/Items/Consumables/Fuel.png)，使用后获得60点开拓力。

在不同的环境使用以获得不同的遗器套装：

- 获取详解：在不同的环境使用会被传送到不同的副本挑战中，击败副本中所有敌人后即可获取2个对应套装的随机遗器（简易模式是4个）
- 副本挑战：副本挑战中不可挖掘或建造，击败所有敌人后会自动退出副本，环境拥有优先级，以下排在前面的环境会覆盖后面的环境

### 套装效果

和崩坏：星穹铁道中一样，本Mod的遗器也都有根据泰拉瑞亚游戏玩法而设计的相对应的套装效果。

#### 隧洞遗器
- **繁星璀璨的天才**（![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Quantum/QuantumHead.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Quantum/QuantumHands.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Quantum/QuantumBody.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Quantum/QuantumFeet.png)）：
  - 二件套：近战伤害提高5%
  - 四件套：盔甲穿透提高4点，当造成近战伤害时，盔甲穿透额外提高4点。

- **戍卫风雪的铁卫**（![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Guard/GuardHead.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Guard/GuardHands.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Guard/GuardBody.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Guard/GuardFeet.png)）：
  - 二件套：免伤提高3%
  - 四件套：当前生命值百分比小于等于50%时，立即回复生命上限4%的生命值，该效果有8秒冷却。

- **熔岩锻铸的火匠**（![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Fire/FireHead.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Fire/FireHands.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Fire/FireBody.png)![隧洞遗器示例](StarRailRelic/Content/Items/Relic/Out/One/Fire/FireFeet.png)）：
